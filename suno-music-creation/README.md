### Module 2: The 5-Layer Prompt Architecture (2 hours)

Master the FrankX prompt engineering system for consistently excellent results.

**The 5-Layer Structure**:

| Layer | Purpose | Example |
|-------|---------|---------|
| **Foundation** | Genre/Style | "Cinematic orchestral hybrid" |
| **Mood** | Emotional landscape | "Powerful and uplifting" |
| **Instrumentation** | Sonic palette | "Soaring strings, epic brass, taiko" |
| **Production** | Mix quality | "Wide stereo, film-quality" |
| **Intention** | Energetic purpose | "Courage to pursue bold dreams" |

**Complete Prompt Formula**:
```
[FOUNDATION], [MOOD], with [INSTRUMENTATION], [PRODUCTION], intention: [INTENTION], [TEMPO/FEEL].
```

**Example**:
```
Ambient electronic meditation, ethereal and transcendent, with crystal singing bowls tuned to 528Hz, soft synth pads, binaural theta waves, gentle wind chimes, spacious cathedral reverb, pristine production, intention: deep inner peace and heart healing, slow and drifting.
```
